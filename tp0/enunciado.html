Tarefa B :: Potenciação

Dado um real x e um inteiro positivo n, pretende-se o cálculo eficiente da potência xn.

Uma estratégia eficiente no cálculo de potências quadradas é o recurso ao squaring (utilização de potencias quadradas). Esta abordagem torna-se particularmente vantajosa quando lidamos com expoentes grandes, uma vez que se  reduz substancialmente o número de multiplicações necessárias. A potenciação por squaring baseia-se no seguinte o resultado:

 

xn = x(x (n-1)/2)2 se n é ímpar; xn = (x n/2)2 caso contrario.

 

Por exemplo, para calcularmos eficientemente x24 , faríamos:

x24 = (x12)2 = ((x6)2)2 = (((x3)2)2)2 = (((((x2*x)2)2)2.

 

Neste caso, realizamos apenas 5 multiplicações em vez de 23.

 

Input

O input consistirá de duas linhas:  a primeira contém o real x e a segunda o inteiro positivo n.

 

Output

Na primeira linha deverá surgir o resultado do cálculo realizado (com duas casas decimais). Na segunda, apresentar-se-á o número de multiplicações efectuadas.

 

 

Exemplo de Input

 

2.0

24

 

Exemplo de Output

 

167777216.00

5

 

Nota: tanto a última linha de input como de output terminam com o carácter de mudança de linha.
